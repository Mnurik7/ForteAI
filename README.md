# ForteAI
РЕЗЮМЕ ПРОЕКТА FORTE.AI
Что представляет собой проект:
ForteAI — платформа с 5 AI-модулями для банковской сферы, построенная на Flask + Google Gemini API.
Страницы и их функции:
Login — авторизация с хешированием паролей
Index — главная страница с обзором всех модулей
SmartAntiFraud — 9 функций:
Быстрый ML анализ
Глубокий AI анализ
Пакетный анализ
История клиента
Статистика
Подозрительные транзакции
Дашборд аналитики
MLOps функции
Реалтайм обработка
AI-Procure — 4 вкладки:
Анализ тендера (извлечение параметров, подбор поставщиков, анализ рисков, отчёты)
Поддержка новичков (рейтинг, инструкции, партнёрства, документы)
Песочница (тренировочные тендеры)
История
AI-Scrum Master — 6 вкладок:
Проекты и команды
Создание задач из текста
Управление спринтами
Анализ встреч
Контроль дедлайнов
Аналитика
AI-Business Analyst — сбор требований:
Интерактивный чат с AI
Генерация BRD
Use Cases
User Stories
BPMN диаграммы
Экспорт документов
Confluence Docs — хранилище документов
AI-Code Review — 6 вкладок:
Анализ архитектуры
Ревью файла
Ревью проекта
Анализ diff
Управление MR
Обучающая поддержка
AI-Ассистент — глобальный виджет на всех страницах
Статистика проекта:
113 API endpoints в app.py
3403 строки в главном файле
5 основных модулей
9 HTML страниц
JSON база данных для хранения данных
ML модели с MLOps функциональностью
